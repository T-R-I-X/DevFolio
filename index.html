About Me

I started coding back in 2014 when I was pretty young. I got into it to distract myself from everything else going on. I started off with JavaScript, which is a go-to language for web browsers. It was a great choice for a beginner like me, and I picked it up really quickly. My first big project was a website I made that runs locally and helps you list out your tasks for the day. It's actually been super helpful for my parents too, funny enough! I really love programming. What started as a hobby has turned into a real passion for me. I get to improve my skills every day, and I'm constantly on the lookout for cool new techniques to add to my coding style. I don't think I'll ever stop learning in this field. It's always evolving, and that's what keeps it so interesting!
